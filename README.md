# saas-pro

## 初始化

请确保已全局安装 yarn

```
npm run init
```

### 本地开发

```shell
# linux macos windows 用户
# 使用dev环境的api
npm run dev

# 使用test环境的api
npm run dev:test

# 使用pre环境api
npm run dev:pre
```
